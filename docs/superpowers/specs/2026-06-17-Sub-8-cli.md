# Sub-8: CLI 智能化 (CLI Enhancement)

**Sprint**: S7
**周期**: 1-2 周
**依赖**: Sub-1 ~ Sub-7
**阻塞**: 无（最终层）

## 目标

将 CLI 从基础 clap 命令行升级为智能交互式：REPL 模式、配置驱动、智能错误诊断、JSON 输出、Shell completions。

## 范围

### In Scope
- REPL 交互模式（`ass2sup> `）
- 智能错误诊断 + 修复建议
- 配置文件 `ass2sup.toml`（已属 Sub-1，CLI 集成）
- JSON 进度输出（脚本集成）
- Shell completions（fish/zsh/bash）
- 插件 CLI 命令

### Out of Scope
- GUI（v2.1+）

## 架构决策

### 双模式 CLI
```rust
pub enum CliMode {
    OneShot { args: Args },
    Repl { config: Config },
}
```

### 智能错误
```rust
pub struct SmartError {
    pub error: Error,
    pub suggestions: Vec<Suggestion>,
    pub docs_link: Option<String>,
}
```

### 配置文件
```toml
# ~/.config/ass2sup/config.toml
[defaults]
fps = 23.976
max_colors = 255
dither = "floyd-steinberg"

[cjk_fallback]
chain = ["Noto Sans CJK SC", "Microsoft YaHei"]

[[style_overrides]]
style = "OP_1"
font = "Source Han Sans CN"
```
